# Week 14: Todo List Application

一個極簡、美觀的待辦事項網頁應用程式 (Todo List App)。

## ✨ 功能特色

- **新增任務**：快速輸入並添加新的待辦事項。
- **狀態切換**：點擊即可標記完成或未完成。
- **資料保存**：使用 LocalStorage，關閉瀏覽器後資料依然存在。
- **清除功能**：一鍵清除所有已完成的項目。
- **響應式設計**：在手機與電腦上皆能完美呈現。

## 🛠️ 使用技術

- **HTML5**：語意化標籤結構。
- **CSS3**：Flexbox 排版、CSS 變數、自定義動畫與現代化 UI。
- **JavaScript (Vanilla)**：原生 DOM 操作與事件處理。

## 🚀 如何執行

1. 下載此專案的所有檔案。
2. 直接點擊 `index.html` 以瀏覽器開啟。
3. 開始使用！

## 📂 檔案結構

- `index.html`: 網頁主結構
- `style.css`: 樣式表
- `script.js`: 程式邏輯
